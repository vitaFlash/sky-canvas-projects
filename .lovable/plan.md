

# DronVista.pro — Coming Soon Landing Page

## Overview
A cinematic, full-screen coming-soon landing page for a certified drone pilot in Valencia, Spain. Apple-inspired minimalism with dark overlay over looping background video.

## Layout (Single Page)

### Full-Screen Hero Section
- **Background**: Looping, autoplay, muted drone video (provided) covering the entire viewport
- **Dark gradient overlay** (semi-transparent black) for text readability
- **Content centered vertically and horizontally**:
  1. Small pilot avatar (circular, provided photo) with subtle glow/border
  2. **Headline**: "Vídeo aéreo con dron en Valencia" — large, bold, white
  3. **Subheadline**: "Piloto de drones certificado creando contenido audiovisual profesional para marcas, eventos y propiedades" — lighter weight, muted white
  4. **Green WhatsApp CTA button**: "Contactar por WhatsApp" — opens wa.me link
  5. **Small muted text below**: "Disponible próximamente · Reserva anticipada abierta"

### Tag Cloud Section
- Below the main content, a row of subtle pill/chip tags in Spanish:
  Inmobiliaria, Eventos, Bodas, Hoteles, Turismo, Publicidad, Redes Sociales, Construcción, Inspecciones, Vídeo Cinemático, FPV Drone, Promoción
- Semi-transparent white borders, minimal style

### Footer
- "dronvista.pro" branding text, small and elegant

## Visual Style
- **Dark cinematic theme** — no light mode needed
- **Typography**: Clean sans-serif, large headline, generous spacing
- **Colors**: White text on dark overlay, green accent for WhatsApp button
- **Animations**: Fade-in on load for text elements, subtle hover effects on tags and button
- **Responsive**: Works beautifully on mobile and desktop

## Assets
- Background video: provided mp4 file
- Pilot photo: provided jpg for circular avatar
